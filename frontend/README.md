# Frontend (React) - Starter instructions

Recommended stack:
- Vite + React + TailwindCSS
- Axios for API calls, React Query for caching

Minimal steps to start:
1. npx create-vite@latest frontend --template react
2. cd frontend
3. npm install
4. npm install tailwindcss postcss autoprefixer
5. Setup API base URL to http://localhost:8000
6. Build pages: Login/Register, File Explorer, Upload component, File preview page
